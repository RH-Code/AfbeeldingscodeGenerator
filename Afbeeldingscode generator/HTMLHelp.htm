<h1>AFBEELDINGSCODE GENERATOR</h1>
 
        <p>Met dit programma kan je de HTML-code genereren die je als Cache-eigenaar nodig hebt wanneer je bv. een banner op je cachelisting wilt plaatsen.<br />
        Aangezien je op de cachelisting op <a href="www.geocaching.com">geocaching.com</a> niet zomaar de HTML-code kan plaatsen omdat er anders als resultaat enkel een afbeelding is te zal zijn,<br />
        moeten we dus die HTML-code omzetten naar code. Daar zorgt dit programma voor.</p>
 
        <h2>URL's</h2>
        
            <h3>Afbeeldingslocatie</h3>
 
            <p>In dit veld typ je de link naar de afbeelding. Dit is een <b>verplicht</b> veld.<br />
            <i>Bv.: http://www.linknaardeafbeelding.be</i></p>
 
            <h3>Omschrijving</h3>
 
            <p>In dit veld typ je de omschrijving van de afbeelding. Dit is een <b>verplicht</b> veld.<br />
            <i>Bv.: geocache</i></p>
 
            <h3>Hyperlink</h3>
 
            <p>In dit veld typ je de link naar de pagina die moet worden geopend wanneer er op de afbeelding wordt geklikt.<br />
            Dit is geen verplicht veld.<br />
            <i>Bv.: http://www.linknaardepagina.be</i></p>
 
            <h3>Hyperlink openen in nieuw tabblad</h3>
 
            <p>Wanneer je dit aanvinkt, zal de pagina waarheen moet genavigeerd worden wanneer er op de afbeelding wordt geklikt,
            openen in een nieuw tabblad of venster.</p>
 
        <h2>Opmaak</h2>
        
            <h3>Breedte</h3>
 
            <p>Hier kan je de gewenste breedte van de afbeelding ingeven. Dit wordt uitgedrukt in pixels (px).<br />
            Dit is niet verplicht. Indien je dit niet doet, wordt de originele breedte van de afbeelding gebruikt.</p>
 
            <h3>Hoogte</h3>
 
            <p>Hier kan je de gewenste hoogte van de afbeelding ingeven. Dit wordt uitgedrukt in pixels (px).<br />
            Dit is niet verplicht. Indien je dit niet doet, wordt de originele breedte van de afbeelding gebruikt.</p>
 
            <h3>Rand</h3>
 
            <p>Hier kan je een rand toevoegen aan je afbeelding door de gewenste dikte van de rand in te geven. Dit wordt uitgedrukt in pixels (px).<br />
            Dit is niet verplicht. Indien je dit niet doet, wordt er geen rand toegevoegd aan de afbeelding.</p>
 
            <h3>Centreren</h3>
 
            <p>Wanneer je hier een vinkje plaatst, zal de afbeelding gecentreerd worden.<br />
            Dit is niet verplicht. Indien je dit niet doet, wordt de afbeelding gewoon links uitgelijnd.</p>
 
        <h2>Uitvoer</h2>
 
            <h3>Maak code knop</h3>
 
            <p>Wanneer je hierop klikt, wordt de code gegenereerd.</p>
        
            <h3>Te kopiëren HTML-code</h3>
 
            <p>Hier komt de code van de code te staan die je als CO (cache owner) nodig hebt om in je listing te plaatsen.<br />
            Je kan deze kopiëren door de tekst te <u>selecteren</u> en met de <u>rechter muisknop</u> te klikken en kiezen voor <u>kopiëren</u>.</p>
 
            <h3>Voorbeeld HTML-code</h3>
 
            <p>Hier krijg je de HTML-code te zien zoals men ze zal zien wanneer de listing klaar is. Dus deze code zal dan door de geocacher kunnen
            gekopieerd worden om op hun profielpagina te plaatsen.</p>

            <p>Hier krijg je een afgewerkt voorbeeld te zien van hoe de afbeelding (banner) eruit zal zien.</p>
